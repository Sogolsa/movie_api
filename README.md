# myFlix Movie API

## Description:

The server-side component of a "movie" web application. The web application will provide users with
access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

## API Endpoint Documentation:

API endpoint documentation is in public folder, in documentation.html, in repository.

## Getting Started

Install dependencies: npm install

## Technologies Used:

- Node.js
- Express.js
- MongoDB with Mongoose
- Postman
- bcrypt
- cors
- express-validator
- jsonwebtoken
- passport
- passport-jwt
- passport-local
